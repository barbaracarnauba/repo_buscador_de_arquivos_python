Buscar arquivos Python (ou de outra extensão) e obter a data da sua última modificação é uma tarefa que pode ser realizada simplesmente através da linha de comando (Prompt de Comando do Windows) a partir do comando dir *.py. Assim, o objetivo deste script foi conversar com o sistema operacional e fazer as buscas através da linguagem Python, pois é muito frequente, em nossos scripts necessitarmos verificar se já existem arquivos, modificá-los, deletá-los etc
